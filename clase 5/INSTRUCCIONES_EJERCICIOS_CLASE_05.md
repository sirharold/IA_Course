## Objetivo general

Practicar:

- consumo de APIs desde Python;
- request y response;
- variables de entorno;
- integracion con modelos;
- validacion de respuestas;
- uso de GitHub Copilot como apoyo de desarrollo.

## Regla comun

En todos los ejercicios deben usar Copilot al menos para:

- bosquejar el script;
- proponer validaciones;
- revisar manejo de errores;
- mejorar la salida;
- o separar configuracion de codigo.
