# Ejercicio 1 - Radiografia de una llamada HTTP

## Objetivo

Entender que se envia y que se recibe al consumir una API desde Python.

## Archivo principal

- `ejercicio1_http.py`

## API usada

- `https://httpbin.org/get`
- `https://httpbin.org/post`

## Lo que deben hacer

- completar o ajustar el script;
- hacer una llamada `GET` con query params;
- hacer una llamada `POST` con JSON;
- imprimir una salida reducida.

## Prompt sugerido para Copilot

```text
Necesito un script Python pequeno que haga una llamada GET con params y una POST con json usando requests.
Quiero imprimir solo status_code, URL final, params recibidos y json reflejado por la API.
No imprimas todo el body.
```
