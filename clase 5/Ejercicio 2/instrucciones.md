# Ejercicio 2 - API publica y salida util

## Objetivo

Consumir una API publica real y convertir la respuesta en una salida breve y usable.

## Archivo principal

- `ejercicio2_meteo.py`

## API usada

- Open-Meteo

## Lo que deben hacer

- revisar el script;
- cambiar coordenadas o variables si quieren;
- dejar una salida resumida en consola;
- agregar una validacion pequena.

## Prompt sugerido para Copilot

```text
Revisa este cliente Python para Open-Meteo.
Quiero que la salida quede reducida a hora, temperatura y probabilidad de precipitacion.
Agrega validaciones pequenas sin sobrecomplicar el script.
```
