## Carpeta principal para alumnos

- `reconciliation_legacy_service`

La idea es que los alumnos abran esa carpeta en Visual Studio Code y trabajen sobre codigo ya preparado para practicar:

- comprension de codigo existente;
- analisis de dependencias y supuestos de datos;
- lectura de logs e incidentes;
- separacion entre hechos, inferencias e incertidumbres;
- propuestas de mejora pequenas;
- diseno de pruebas desde riesgos observados;
- consultas SQL de investigacion;
- una mini tarea ETL local;
- artefactos visuales en Mermaid.
