# Ejercicios - Clase 3

## Foco actualizado

Esta clase ya no se centra solo en "explicar archivos". Ahora busca que los alumnos produzcan salidas mas utiles:

- mapa del flujo;
- flujograma Mermaid;
- diagrama de dependencias;
- secuencia visual del incidente;
- mapa de evidencia;
- consultas SQL de investigacion;
- evaluacion de scripts pequenos de correccion de datos;
- comprension de una mini tarea ETL;
- propuesta de mejora pequena;
- pruebas derivadas desde riesgos.

## Ruta de trabajo

Los alumnos deben trabajar en:

- `reconciliation_legacy_service`

Y seguir la guia principal:

- `Ejercicios/guia-ejercicios-clase-3.md`

## Ejercicios nucleares recomendados

1. Mapa del flujo principal.
2. Flujograma visual en Mermaid.
3. Diagrama de dependencias.
4. Triage de incidente.
5. SQL de investigacion.
6. Mini tarea ETL.
7. Pruebas desde riesgos.

## Ejercicios de profundizacion

1. Secuencia visual del incidente.
2. Hechos versus inferencias.
3. Mapa de evidencia.
4. Script de correccion de datos.
5. Workflow por roles.

## Criterio docente

- pedir evidencia minima en cada respuesta;
- privilegiar artefactos visuales cortos sobre texto excesivo;
- incluir al menos una actividad de datos cercana al trabajo real del alumno;
- si aparece ETL en el temario, usar el script local para mostrar extract-transform-load sin meter infraestructura adicional;
- usar Mermaid antes que herramientas mas pesadas;
- no introducir agentes formales, MCP ni skills como requisito del alumno;
- mantener el foco en comprension y mantenimiento de sistemas existentes.
