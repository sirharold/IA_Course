# AGENTS.md

## Objetivo del repo

Este proyecto existe para practicar explicacion de codigo, documentacion, refactorizacion pequena e instrucciones persistentes con asistentes de IA.

## Como usar este repo con agentes

- Revisa primero `.github/copilot-instructions.md`.
- Revisa luego los archivos en `.github/instructions/` para contexto separado por ambito.
- Si una instruccion entra en conflicto, prioriza:
  1. seguridad;
  2. requerimientos visibles del repo;
  3. arquitectura y datos;
  4. estilo y documentacion.

## Ambitos separados en este ejemplo

- arquitectura
- requerimientos
- seguridad
- pruebas
- datos
- refactorizacion
- documentacion
- operaciones

## Regla clave

No inventes reglas de negocio no visibles en el codigo o en `data/sample_reviews.json`.
