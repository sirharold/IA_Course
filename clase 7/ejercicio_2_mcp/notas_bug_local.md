Problema observado:

El script de conciliacion de transacciones demora demasiado cuando se procesa el archivo completo de gateway.

Sintomas:

- la corrida completa tarda varios minutos;
- no queda claro en que parte del flujo se concentra la demora;
- el equipo sospecha que hay reprocesamiento innecesario;
- algunos usuarios creen que el problema empeoro despues del ultimo ajuste.

Impacto:

- el reporte sale tarde;
- cuesta validar si el problema es de datos o de implementacion;
- el equipo necesita abrir un issue tecnico para investigar.
